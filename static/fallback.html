Key Features

Transaction Flow Visualization

Interactive flowcharts showing transaction paths between wallets, protocols, and other entities on the Solana blockchain.

Wallet Analysis

Detailed analysis of wallet activity, funding sources, and connection patterns to identify relationships and behaviors.

Entity Labeling

Identification of known exchanges, DEXs, projects, and other entities to provide context for transaction flows.

Transaction Timeline

Chronological view of transaction activity with filtering capabilities by date, volume, and transaction type.

Transaction Clustering

Identification of related transactions and unusual transaction patterns for deeper analysis.

Critical Path Highlighting

Automated detection of critical transaction paths between wallets based on volume, frequency, and other factors.
